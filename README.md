# Consent Guardrails

A real-time consent compliance dashboard for monitoring the health of AI projects.

## Features

* Displays total projects, compliance status, and recent violations
* Metrics are refreshed every 5 minutes
* Users can drill down into individual project logs

## Requirements

* Python 3.8 or later

## Installation

1. Clone the repository
2. Install dependencies using `poetry install`
3. Run tests using `pytest`

## Usage

1. Run the dashboard using `python -m consent_guardrails`
2. Drill down into project logs using `python -m consent_guardrails --project-id <id>`
